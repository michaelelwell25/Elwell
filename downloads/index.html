Individual files

The package files committed to this repository (commit 6adc82f):

  • ebdid.ado — Stata program file
  • ebdid.sthlp — Stata help file (SMCL)

Download ebdid.ado   Download ebdid.sthlp

Installation (Stata)

  1. Download ebdid.ado and ebdid.sthlp from the links above.
  2. Find your PERSONAL ado directory in Stata: in Stata type sysdir and note the path labeled PERSONAL.
  3. Copy the files into that directory. Example (macOS / Linux): 
    cp ~/Downloads/ebdid.ado ~/ado/personal/
cp ~/Downloads/ebdid.sthlp ~/ado/personal/
  4. Restart Stata and confirm the help file is available: help ebdid. Run an example to check everything works.

Example usage

// in Stata
sysdir
// copy files to PERSONAL (example path)
help ebdid
ebdid y, group(state) time(year) treated(D)
